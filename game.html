Help

Welcome to StarryNightAdventure. Here's some useful commands for you to use:

General commands

  • 'replay' : Replay chapters you have reached
  • 'quit' : Quit current game
  • 'help': Show instructions
  • 'inventory': Show items you currently have
  • 'map': Show the map of areas you have explored

Movement commands

  • 'go north/west/south/east': Move to different areas, if there is a path or unlocked door
  • 'enter room I/room II/shop': Enter different rooms, can only be used in the corridor.

Movement commands can only be used if character is not examining / interacting / talking. Movement commands can be used via movement buttons above or access keys (UP, DOWN, LEFT, RIGHT).

Object commands

  • 'get *item name*': Get an item
  • 'examine *item name*': Examine an item
  • 'interact *object name*': Interact with an ungettable object
  • 'use *item name*': Use an item when interacting
  • 'answer *input*': Answer a puzzle / riddle when interacting
  • 'back': Stop interacting / examining

Character commands

  • 'talk': Talk to a character
  • 'choose': Choose a dialog option
  • 'use *item name*': Use an item when talking

Use CTRL + Z to close popup screen.
